# hw8
вариант 3 
Описание задачи дробного рюкзака с жадным алгоритмом
Задача: есть предметы с весом и стоимостью, рюкзак задан по вместимости. Нужно выбрать предметы или их части так, чтобы суммарная стоимость была максимальной при ограничении по весу.

Жадный алгоритм:

Считаем удельную стоимость для каждого предмета: value/weight.

Сортируем предметы по удельной стоимости по убыванию.

Последовательно берем предметы или их часть в рюкзак, пока есть место.
Временная сложность
Сортировка предметов по удельной стоимости занимает O(nlogn), где n — число предметов.Почему такая сложность
Основная затратная операция — сортировка. Сам выбор предметов — простой проход по отсортированному массиву за линейное время.
Коэффициент аппроксимации
Для дробного рюкзака жадный алгоритм дает оптимальное решение, то есть коэффициент аппроксимации равен 1. Это значит, что решение жадного алгоритма максимально возможное и не отличается от оптимального по стоимости.



контрольный вопрос 3
PTAS и FPTAS — это алгоритмы для приближённого решения сложных задач, которые нельзя быстро решить точно.

PTAS — алгоритм, который может найти решение, близкое к оптимальному, насколько скажешь (зависит от параметра ε), и при этом время работы растёт полиномиально по размеру задачи, но может сильно увеличиваться при стремлении к очень маленькому ε (то есть в зависимости от ε время может расти очень быстро).

FPTAS — это особый случай PTAS, где время работы растёт полиномиально и от размера задачи, и от самого параметра ε (точнее, от 1/ε). Это значит, что уменьшать ε (повышать точность) в таком алгоритме можно, и алгоритм всё ещё будет работать относительно быстро.

Проще говоря: PTAS может замедлиться, если хочешь очень точный ответ, а FPTAS гарантирует, что и при очень высокой точности алгоритм будет работать эффективно.

FPTAS — это более "удобный" и быстрый в плане управления точностью вид PTAS.
